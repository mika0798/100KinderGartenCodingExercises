# Easy Level Coding Challenges
### 1. Sum of digits of a number
```
Given a number n, find the sum of its digits.
```
### 2. Reverse digits of a number
```
Given an Integer n, find the reverse of its digits.
```
### 3. Prime numbers
```
Given a positive integer, check if the number is prime or not. A prime number is a natural number
greater than 1 that has no positive divisors other than 1 and itself. Examples of the first few
prime numbers are {2, 3, 5, ...}
```
